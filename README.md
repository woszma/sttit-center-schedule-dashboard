# STTIT - 中心課表dashboard

單頁 dashboard，顯示目前日期、時間，以及從 Google Calendar 載入的當日課表。

## 功能

- 即時日期與時間
- 當日 Google Calendar 行程
- 顯示事件標題、時間、地點
- 可在頁面內輸入並儲存 `Google API Key` 與 `Calendar ID`

## 使用方式

直接開啟 `index.html` 即可。

若要成功抓取 Google Calendar：

1. 準備可用的 `Google API Key`
2. 準備 `Calendar ID`
3. 將日曆設為公開，或確認 API 權限允許存取
